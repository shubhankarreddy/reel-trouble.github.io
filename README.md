# Reel Trouble 🎮

Built this as a small experiment to make learning a bit addictive

## Idea
Students lose focus fast. So tried adding:
- quick feedback
- scoring system
- game feel

Basically a small dopamine loop to keep them engaged.

## What it does
You interact → get instant response → score updates

Simple, but keeps you hooked.

## Use case
- classroom practice
- quick revision
- self-learning
